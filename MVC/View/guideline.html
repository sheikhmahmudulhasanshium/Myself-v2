Guidelines

The information must meet the following requirements:

  • First name must contain atmost 20 character
  • Last name must contain atmost 20 character
  • User name
    • must be unique
    • must contain atmost 30 character
    • must contain no space among the characters
    • can be a combination of character and number eg: hp_123
  • Email adress
    • must be unique
    • must be small case
    • must contain the special character '@'
    • must contain a domain eg: 'gmail.com', 'yahoo.com', etc.
    • must start the prefix with an alphabetical character
    • prefix can be a combination of character and number eg: 'hp1_1'
    • must not contain special characters at the begining and 2 or more times consecutively eg: '_ha', 'emdad..'
  • Password
    • must contain atleast 8 characters
    • must contain atmost 20 characters
    • must contain a capital letter
    • must contain a lowercase letter
    • must contain a number
    • can contain special characters eg: '_', '@', '!', etc.
